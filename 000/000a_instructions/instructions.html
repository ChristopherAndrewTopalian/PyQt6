Creating Python Apps with PyQt6

How to make applications using Python and PyQt6

Download Python

https://www.python.org/downloads

Install Python

Double Click on the Python Install file and make sure you "Add to Path", when installing.

Open Windows Command Prompt

Search Windows for Command Prompt and open it

Install PyQt6 by Typing in Windows Command Prompt

pip install PyQt6

then Press Enter

Make a New File in VSCode

Control + N to make a New Text File
Type the script named 001a.pyw into the New Text File
Control + S to Save the File. Name the file anything you want but, make sure to use .py or .pyw as the file extension type, such as 001a.pyw

Saving Our First Python File

When we save a Python file, we can choose .py or .pyw for the extension type.
If we use ourApp.py, it will open a console window when we start our app.
If we use ourApp.pyw, it will NOT open a console window when we start our app.

VSCode

When programming using Python and VSCode, there are many useful extensions for the Python language to make things easier.
Make sure to install the Microsoft extension named Python.

Run the Python File

Double Click on the Python file to start the application or
if you installed the extension Python from Microsoft in VSCode, then you can press the play button at the top right of the VSCode editor to start the Python application.

When you are ready, move on to 002a.pyw

Enjoy the many examples to learn from.
Remember, you can always add more examples anytime by pressing Control + N to make a New Text File.
Happy Scripting :-)
